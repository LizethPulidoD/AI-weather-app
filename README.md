# 🌤️ Weather App

Aplicación web sencilla que permite consultar el clima actual o el pronóstico de 5 días de una ciudad, utilizando la API de Open-Meteo.

---

## 🚀 Funcionalidades

- 🔍 Buscar una ciudad por nombre
- 🌡️ Ver el clima actual
- 📅 Consultar pronóstico de 5 días
- ❌ Manejo de errores (ciudades inválidas, problemas de red, etc.)
- 🌐 Interfaz web simple y fácil de usar


---

## ⚙️ Instalación

1. Clona o descarga el proyecto

2. Instala dependencias:
npm install

## Cómo ejecutar
🔹 Opción 1: Aplicación web
node server.js

Luego abre en tu navegador:

http://localhost:3000

## Ejemplo de uso
Ingresa una ciudad (ej: Bogotá)
Selecciona:
Clima actual
Pronóstico de 5 días
Visualiza los resultados
